Morse: 
  Binary version: 
    0: space inter element character 
    000: space between letters 
    000 0 000: space between words
    1: dot 
    111: dash

    A = *_   ->  1 0 111  
    B = _*** -> 111 0 1 0 1   
    EU = * **_ -> 1 000 1010111

1. ABSTRACAO
  A mensagem m contem com certeza uma frase completa(não necessariamente coerente), sem erros ortograficos ou de perda de dado e
sem espacos.

Exemplos:

CORRETOS
 a. "AMARÉVIDA"
 b. "AMORAPESSOA" note que é "AMOR" e não "AMORA" o algo deve pegar a priemeira ocorrencia
 c. "AAULADEECONOMIADEQUARTAÉTOPDEMAIS"
 d. "UMCOPODECAFÉDEMANHÃOUUMTIRÃONACABEÇA"

ERRADOS
  a. "AMAR É VIDA" contem espacos
  b. "AMOVC"  contem abreviacoes
  c. "SHEREKÉVIDA" contem nome proprio

2. OBJETIVO
  Identificar se a mensagem contem apenas palavras, para isso encontrar uma palavra, bater no dicionario de lingua portuguesa e se tiver uma entrad nele, addicionar a palavra na string que representa a mensagem final e continuar procurando outras palavras na string original até chegar ao seu final.

3. SUGESTAO DE ALGORITMO
  Andar letra por letra até encontrar uma palavra, quando achar, remover os caracteres encontrados da mensagem original e adicionar a palavra encontrada na string que representa a mensagem final "decodificada".

1. ESTRUTURA DO BANCO
Registro: parte da mensagem que foi entregue para decodificacao, cada registro é composto pelos seguintes atributos

Atributos:
  1 Timestam recebimento
  2 pedaco da mensagem original 
  3 nó worker para qual foi enviado 
  4 tipo de cyfra que esta sendo tentada 
  5 timestamp de envio para o noh, para que se possa contar quanto tempo esse noh esta tentado essa cifra
  6 mensagem decodificada que sera retornada pelo trabalhador

registro  timestamp de |  pedaco da mensagem  |   nó worker para  |  cifra  | timestamp de envio | mensagem 
          recebimento  |           original   |  qual foi enviado |         |  (conter statusa)  | decodificad
